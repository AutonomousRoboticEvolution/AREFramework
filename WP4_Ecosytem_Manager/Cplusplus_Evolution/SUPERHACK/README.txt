(1) Put these file in your VREP files directory
(2) Change the individual number after "ind: " to the individual number you would like to load
(3) Look up the phenotype file of this individual (NOT THE GENOTYPE!!!!) and put it into the morphologies200 folder. 
(4) Run vrep with the arguments "-g200 -g7 -gfiles"
(5) Press start and see your individual be created! You should now be loading the individual you want.
